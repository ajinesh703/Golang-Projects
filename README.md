## 🚀 Projects

### 1. 🐣 First Step
> **Category:** Beginner | Getting Started with Go

An introductory project covering the fundamentals of the Go programming language. Perfect for anyone just starting out with Go syntax, types, and basic program structure.

**Topics covered may include:**
- Variables, constants, and data types
- Control flow (if/else, loops)
- Functions and error handling
- Basic I/O operations

**Run it:**
```bash
cd "First Step"
go run main.go
```

---

### 2. 📷 QR Generator
> **Category:** Intermediate | CLI Tool

A command-line QR code generator written in Go. Takes a string or URL as input and outputs a QR code image (PNG).

**Features:**
- Generate QR codes from any text or URL
- Saves output as a PNG image
- Simple and fast CLI usage

**Run it:**
```bash
cd "QR Generator"
go mod tidy
go run main.go
```

## 🛠️ Prerequisites

- [Go 1.18+](https://go.dev/dl/) installed on your system
- Basic familiarity with the terminal

**Verify your Go installation:**
```bash
go version
```

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/ajinesh703/Golang-Projects.git
   cd Golang-Projects
   ```

2. **Navigate to a project folder**
   ```bash
   cd "QR Generator"
   ```

3. **Install dependencies (if any)**
   ```bash
   go mod tidy
   ```

4. **Run the project**
   ```bash
   go run main.go
   ```
